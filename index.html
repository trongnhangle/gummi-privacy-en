PRIVACY POLICY

NOVA UNIVERSE SOFTWARE COMPANY LIMITED

Last Updated: November 26, 2024

1. INTRODUCTION

1.1 Privacy Commitment

NOVA Universe is committed to protecting user privacy and information security with the following principles:

  • Transparency in handling information
  • Respect for individual privacy
  • Ensuring data security with modern technical measures
  • Compliance with international security standards

1.2 Scope of Application

This policy applies to all users of the Gummi app, including:

  • Direct users
  • Visitors
  • Registered service users
  • Users utilizing AI features

2. INFORMATION COLLECTED

2.1 Types of Information

Gummi is designed with the principle of maximum protection of personal information. We:

2.1.1 Do Not Collect Directly
  • Do not collect user names
  • Do not request email addresses
  • Do not collect phone numbers
  • Do not require demographic information
2.1.2 Do Not Track
  • Do not record geographic locations
  • Do not store search history
  • Do not track user behavior

2.2 Potential Exceptions

While we do not actively collect information, some data may be recorded:

  • IP address when connecting
  • Type of accessing device
  • Operating system used
  • Access time

3. DATA PROCESSING AND USE

3.1 Purpose of Use

User data is used for one purpose only: Improving Advertising Experience

3.1.1 Improvement Strategies
  • Analyzing user trends
  • Optimizing interfaces
  • Adjusting appropriate advertising content
  • Enhancing service quality

3.2 AI Data Processing

3.2.1 Processing Source
  • Uses third-party professional APIs
  • Does not develop internal AI systems
3.2.2 File Management
  • Files uploaded by users (images, PDFs) are processed immediately
  • Do not store permanently
  • All data will be deleted after processing

4. SECURITY AND SAFETY

4.1 Technical Measures

  • Use HTTPS protocol
  • Encrypt connections
  • Protect against network attacks
  • Strict access control

4.2 Compliance with Standards

  • Meets international security regulations
  • References standards such as GDPR, CCPA

5. USER POLICY

5.1 Age Limitation

  • For users aged 17 and older only
  • Strict age access controls
  • Block and remove violating accounts

5.2 User Rights

Since we do not collect personal information, we:

  • Do not have a process for requesting detailed data deletion
  • Commit to transparency regarding non-storage of personal information

6. DEPLOYMENT SCOPE

6.1 Geography

  • Does not provide services in China
  • Complies with the laws of each country
